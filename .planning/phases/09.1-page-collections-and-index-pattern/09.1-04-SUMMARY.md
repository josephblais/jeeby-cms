## Decisions Made

- Eager listPages on open (not submit): populates parent picker immediately and avoids extra network call on submit. Cancellation flag prevents stale state if modal closes mid-fetch.
- CSS scope: `.jeeby-cms-slug-prefix` at line 901 is already used by PageManager slug row. New styles scoped under `.jeeby-cms-slug-prefixed .jeeby-cms-slug-prefix` to prevent cascade.
- Object spread for conditional fields: `...(pageType === 'collection' ? { isCollectionIndex: true } : { parentSlug: parentSlug || null })` satisfies the test regex that checks for colon syntax.
